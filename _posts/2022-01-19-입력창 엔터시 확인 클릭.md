### 입력창 엔터시 확인 클릭
***
입력창 엔터시 확인버튼 클릭되도록 하는 코드  
`onkeyup="enterkey();`  
```
<input type="password" name="checkPassword" maxlength="20" title="비밀번호" placeholder="비밀번호를 입력하세요." onkeyup="enterkey();">  
```

13번이 엔터이다.  
if 문으로 확인해주지 않으면 키를 입력하는 것 모두 확인버튼으로 실행되어 버린다.  
```
<script>
function enterkey() {
    if (window.event.keyCode == 13) { 
    	passwordCheck();
    }
}
</script>
```
